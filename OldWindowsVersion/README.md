
# BlazingMandelbrot

***Version 1.1*** 

Released February 27 2021 

Program that displays a cool, fiery version of the Mandelbrot fractal. 

Uses [SFML](https://www.sfml-dev.org/) and the [ImGui](https://github.com/ocornut/imgui) libraries.
